finding elements in elements & CSS

<p id="testp1">
    test
    <span>red</span>
    <span>green</span>
    <span class="myclass">blue</span>
</p>
<script>
const myElement = $4("#testp1");
myElement.find("span")[0].css({color: "red"});
myElement.find("span")[1].css({color: "green"});
myElement.find("span.myclass").css({color: "blue"});
</script>

result:

test red green blue
